# Laughter Recognition Models

Complete code for *"Cross-Corpus Generalization Failure of Laughter Recognition Models and Its Implications for Psychological Wellbeing Assessment"* (Scientific Reports, ID: 32fcfbe0-9aa4-4936-a4f0-08c75dfa78e6).

## Quick Reproduction

1. `pip install -r requirements.txt`
2. Update dataset paths in `train10.py` (lines 15-18)
3. `python train10.py`
4. View figures in `outputs/`

**Key outputs reproduced**:
- EXP1: Accuracy 0.99±0.002
- EXP2: Detection rate 0.28±0.04  
- EXP3: Detection rate 0.97±0.01
- t-SNE visualization
